`OpenClaw` 有两种可以直接打开的“客户端”形态：

1. Web 控制台
```bash
openclaw dashboard
```
或者直接在浏览器打开：
```text
http://127.0.0.1:xxxxx/
```

2. 终端 UI

```text
openclaw tui
```
